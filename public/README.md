## üì∏ Required Images

### Hero Image
Place your hero background image at: `hero/hero.jpg`
- Minimum dimensions: 1920x1080px
- Format: JPG or PNG
- This will be the main background image on the homepage

### Product Images
Each product needs 3 images named `01.jpg`, `02.jpg`, `03.jpg`:

**Hoodies:**
- `products/hoodies/hoodie-01/` - Essential Oversized Hoodie
- `products/hoodies/hoodie-02/` - Archive Logo Hoodie
- `products/hoodies/hoodie-03/` - Technical Zip Hoodie

**Pants:**
- `products/pants/pant-01/` - Carpenter Cargo Pants
- `products/pants/pant-02/` - Tailored Track Pants

**T-Shirts:**
- `products/tshirts/tshirt-01/` - Core Oversized Tee
- `products/tshirts/tshirt-02/` - Graphic Print Tee

### Model Images
Each product needs 2 model images named `01.jpg`, `02.jpg` in the corresponding folder under `models/`

- **Format**: JPG (optimized) or PNG
